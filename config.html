      Guia rapida

      1
      Define los datos generales

      Actualiza el titulo del tablero y los roles autorizados. Usa una linea por usuario.

      2
      Configura tus contenedores

      Cada contenedor agrupa tarjetas relacionadas. Agrega los widgets que necesites y define el topic MQTT.

      3
      Guarda y publica

      Revisa el resumen y presiona Guardar en servidor. Puedes descargar o importar configuraciones previas.

      Buenas practicas

      • Utiliza nombres descriptivos para encontrar contenedores rapidamente.
      • Duplica objetos para reciclar configuraciones y ajustar solo lo necesario.
      • Descarga la configuracion antes de cambios grandes para mantener un respaldo.
