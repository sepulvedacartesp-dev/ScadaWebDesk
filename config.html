      Guia rapida

      1
      Define los datos generales

      2
      Configura tus contenedores

      Cada contenedor agrupa tarjetas relacionadas. Agrega los widgets que necesites y define el topico.

      Buenas practicas

      • Utiliza nombres descriptivos para encontrar contenedores rapidamente.
      • Duplica objetos para reciclar configuraciones y ajustar solo lo necesario.
      • Descarga la configuracion antes de cambios grandes para mantener un respaldo.
